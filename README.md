# React Native Simple Login Template

## How to use?

1. Download or clone this repo.

2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project on iOS / Android or Web.

```js
 expo start --ios
 // or
 expo start --android
 // or
 expo start --web
```
